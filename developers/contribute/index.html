Contribute

Contribute or use our software to build your projects

With recent discussion about PRIMS, Boundless Informant, and FISA orders, there have been increased inquiries about getting involved with open source communication apps that are secure & private.

Give Feedback

Use the software and provide feedback. One of the best ways to help is by providing valuable user feedback. Install our apps and file bugs you encounter or features you feel are missing on the issue tracker.

Translation

Translate to non-English languages. Localizing the software for different regions helps ensure we can distribute it as widely as possible. You can help with translation by joining the Transifex translation teams for our apps.
